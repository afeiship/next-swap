# next-swap
> Array swap for next.

[![version][version-image]][version-url]
[![license][license-image]][license-url]
[![size][size-image]][size-url]
[![download][download-image]][download-url]

## installation
```bash
npm install -S @jswork/next-swap
```

## usage
```js
import '@feizheng/next-swap';

const arr = [1, 2, 3, 4];
nx.swap(arr, 0, 3);

// [4, 2, 3, 1]
```
